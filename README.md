# BookStream


<h1>书非借而不能读</h1>

这个微信小程序功能主要包括：添加自己的书籍，借出及借入功能，浏览书籍，评价书籍，以及相应的排行榜

clone这个项目到本地，修改config.js里的api地址，使用自己的appid，即可查看效果
