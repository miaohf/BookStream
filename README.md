<h1>书非借而不能读</h1>

这个微信小程序功能主要包括：
<ul>
	<li>添加自己的书籍</li>
	<li>借出及借入功能</li>
	<li>浏览书籍</li>
	<li>以及相应的排行榜</li>
</ul>

clone这个项目到本地，修改config.js里的api地址，使用自己的appid，即可查看效果
